# AI Expense Tracker

[![CI](https://github.com/chrultom/ai_expense-tracker/actions/workflows/ci.yml/badge.svg)](https://github.com/chrultom/ai_expense-tracker/actions/workflows/ci.yml)

**Pure vibe coding: Not a single line was written manually.**
This project is an experimental implementation where 100% of the code was generated by AI (Roo Code).

## Features
- 💰 **Expense tracking** with easy categorization.
- 📅 **Monthly view** with intuitive navigation.
- 📊 **Financial summaries** and data visualization using Recharts.
- 🗄️ **SQLite database** integration with Prisma ORM.
- 🎨 **Modern UI** built with shadcn/ui and Tailwind CSS v4.
- 📱 **Mobile First** responsive design.

## Technology Stack
- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Database:** [Prisma](https://www.prisma.io/) + SQLite
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Charts:** [Recharts](https://recharts.org/)
- **Icons:** [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Version 18.17 or higher recommended)
- [npm](https://www.npmjs.com/) (included with Node.js)

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/chrultom/ai_expense-tracker.git
   cd ai-expense-tracker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Copy the example environment file to create your local `.env` file:
   ```bash
   # Unix/macOS
   cp .env.example .env

   # Windows
   copy .env.example .env
   ```
   *(The default configuration uses a local SQLite database named `dev.db` in the `prisma` folder.)*

4. **Initialize Database & Start Development Server:**
   The `dev` script is configured to automatically synchronize your database schema:
   ```bash
   npm run dev
   ```
   This command will:
   - Push the Prisma schema to your local SQLite database.
   - Generate the Prisma Client.
   - Start the Next.js development server at [http://localhost:3000](http://localhost:3000).

5. **Open the App:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your web browser.
